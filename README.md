# AVIVO Frontend Task

## 🚀 Setup Instructions

1. Clone the repository:


2. Navigate to project folder:
cd avivo-frontend-task

3. Install dependencies:
npm install

4. Run the application:
npm run dev

5. Open in browser:
http://localhost:5173

---

## 📌 Features Implemented

- Fetch users from API (https://dummyjson.com/users)
- Display user list (name, company, role, country)
- Search users across multiple fields
- Add user (local state only)
- Delete user (local state only)
- Refresh user list from API

---

## 🛠️ Tech Stack

- React (Vite)
- Axios
- Chakra UI
- React Icons

---

## ⚠️ Notes

- Add and Delete operations are handled locally (no backend interaction)
- Refresh resets data from API
